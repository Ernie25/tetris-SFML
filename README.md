# tetris ะก++
Simple Tetris game written as a course work in C++.
The main objective was to use principles of OOP.
SFML(Simple and Fast Multimedia Library) was used for graphical illustration.


# Controls
- `<-` Move left
- `->` Move right
- `arrow up` Rotate figure
- `arrow down` Speed Up
